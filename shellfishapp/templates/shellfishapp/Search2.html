A Convincing Collection and Catalogue of Marine Organisms Reported from Indian Waters

ShellFishRIS pronounced as ShellFish Resource Information System of India integrated with data and user management capabilities aims to provide a much convincing and comprehensive list of names of marine organisms reported from Indian waters including information on synonymy, type specimen, distribution, biology and specimen occurrence. For developing the system, data from authoritative published sources like online databases and literatures published in reputed journals and books were collected compiled and screened before population the data in the database of the information system. The highest priority goes to valid names and other names which have been included so that this system serves as a register to serve as a guide to interpret taxonomic literature. ShellFishRIS has been developed using MysQL for backend database development and Django web framework for developing the web- based information system comprised of web interfaces and tools integrated with the database facilitating the user interactive web pages for browsing the information and data submission into the system by the authenticated users through their login credentials after registration.

Resources to build this information system was provided by ICAR- National Bureau of Fish Genetic Resources Lucknow under the institute funded project entitled “Information System of selected commercially important shellfish resources from prioritized Indian aquatic ecosystem” under mandate “Cataloguing and classification of fish genetic resources”.
